# Student Management System (Spring Boot MVC)

This project implements a Student Management System using:

- Spring Boot
- MVC Architecture
- Hibernate (JPA)
- H2 Database
- Thymeleaf

## Features

- Add student
- View student list
- Prevent duplicate email
- Data stored in database
- MVC architecture implemented

## Tech Stack

- Java 17
- Spring Boot
- JPA / Hibernate
- H2 Database
- Maven

## Run the Project

```bash
mvnw spring-boot:run
```

Open:

```
http://localhost:8080
```
